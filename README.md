# ApexCleaning 🚗

A modern, responsive car detailing website built to showcase services, capture bookings, and provide a clean, professional experience for customers.

---

## 🌐 Features

- 📱 Fully responsive design (mobile + desktop)
- 🧼 Service packages (Silver, Gold, Premium Add-ons)
- 📝 Booking form with real-time email delivery (EmailJS)
- ⭐ Customer reviews section
- ⚡ Fast, static website (no backend required)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- EmailJS (client-side email service)

---

## 📩 Booking System

The booking form uses EmailJS to send customer requests directly to your email without needing a backend server.

**Flow:**
1. Customer fills out booking form  
2. Form data is sent via EmailJS  
3. Booking request is delivered instantly to your inbox
